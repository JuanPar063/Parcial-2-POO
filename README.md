# Parcial 2 - Programación Orientada a Objetos (POO)

Entrega del segundo parcial de la materia **Programación Orientada a Objetos**, que incluye el diagrama UML de la solución, la implementación en Python y la documentación del programa.

## Descripción

Este proyecto es la entrega del parcial 2 de la materia POO. Contiene el diseño del sistema mediante un diagrama UML, su implementación en Python siguiendo los principios de la programación orientada a objetos y un documento de documentación formal del programa.

## Contenido del Repositorio

| Archivo | Descripción |
|---|---|
| `Parcial 2/parcial_2_poo.py` | Implementación del programa en Python |
| `Parcial 2/Parcial 2 (UML).png` | Diagrama UML de la solución |
| `Documentación Parcial 2 (POO).docx` | Documento de documentación del programa |

## Diagrama UML

El diagrama UML muestra la estructura de clases del sistema, incluyendo atributos, métodos y las relaciones entre clases (herencia, composición, asociación).

## Tecnologías Utilizadas

- **Python** – Implementación de la solución
- **Programación Orientada a Objetos** – Paradigma de programación aplicado
- **UML** – Lenguaje de modelado unificado para el diseño del sistema

## Principios POO Aplicados

- **Encapsulamiento** – Atributos y métodos privados/públicos
- **Herencia** – Jerarquía de clases
- **Polimorfismo** – Comportamiento diferenciado según el tipo de objeto
- **Abstracción** – Modelado del problema mediante clases abstractas
